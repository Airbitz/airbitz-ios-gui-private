Settings

Bitcoin Denomination: Change the view of bitcoin values throughout the app. (Default: mBitcoin)

Account: Change your account password, withdrawal PIN or recovery questions and answers.

Name: Optionally enter your first name, last name, and/or nickname. These fields will be sent to users that you request bitcoin from. This information is never known by Airbitz or any parties other than those who you request bitcoin from.

Options: Choose settings for Auto-Logout or Default Currency, and change the list of transaction categories that appear in the Transaction Details.

The "Default Currency" is the fiat currency used to view wallet balances in the "Wallets" screen. This does not affect the fiat currency used to display transaction amounts in wallets. Each wallet has a fiat currency tied to it upon creation.

Default Exchange: Choose the exchange service to use to determine the exchange rate of bitcoin with various fiat currencies.
